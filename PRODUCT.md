# Product

## Register

product

## Users

Implementation project managers, delivery managers, and project owners working inside enterprise delivery teams. They use the product primarily as a Windows/macOS desktop application during active customer delivery to manage tasks, risks, deliverables, weekly reports, and the overall project control process.

## Product Purpose

Implementation PM helps project managers keep enterprise implementation work visible, structured, and under control in a desktop-first workspace. It centralizes project progress, task execution, scope, risk and issue tracking, deliverable acceptance, weekly reporting, and AI-assisted generation of delivery artifacts. Success means a project manager can quickly understand project health, decide what needs attention, and produce consistent delivery outputs without scattering work across spreadsheets, chat threads, and ad hoc documents.

## Brand Personality

Efficient, technical, intuitive. The interface should feel calm, capable, and easy to learn, with clear operational affordances and low friction for repeated daily use.

## Anti-references

Avoid marketing-style SaaS pages, decorative dashboards, excessive hero layouts, dense spreadsheet replicas without hierarchy, and visually clever interactions that slow down routine project work. The product should not feel like a generic admin template, a toy AI demo, or a heavy enterprise system that hides simple workflows behind too many controls.

## Design Principles

1. Make project status scannable before it is editable.
2. Keep repeated operations close to the data they affect.
3. Preserve manager confidence with explicit states, confirmations, and recoverable actions.
4. Let AI-assisted generation feel like part of the delivery workflow, not a separate novelty.
5. Favor compact clarity over decorative spaciousness.

## Accessibility & Inclusion

Use standard product accessibility expectations for desktop apps: keyboard-accessible controls, visible focus states, sufficient color contrast, reduced-motion support, and color-blind friendly status communication that does not rely on color alone.
